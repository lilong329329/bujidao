### 1、电脑翻墙之后，出现无法pull 和 push代码问题解决

```java
问题：
 Failed to connect to github.com port 443:connection timed out
解决：
1、查询是否使用代理：git config --global http.proxy 
2、取消全局代理：
git config --global --unset http.proxy
git config --global --unset https.proxy
 
```
